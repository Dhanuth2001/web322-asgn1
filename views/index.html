About This Project

The Wildlife Sighting Tracker is a backend-focused web application developed using Express.js. It provides a REST-style API that allows users to retrieve, filter, and search wildlife sighting data stored in a JSON file.

Available API Endpoints

Click on any endpoint below to view the JSON response.

GET /api/sightings

Retrieve all wildlife sightings
GET /api/sightings/verified

Retrieve only verified sightings
GET /api/sightings/species-list

View a list of unique species names
GET /api/sightings/habitat/forest

View all forest habitat sightings
GET /api/sightings/search/eagle

Search for the first eagle sighting
GET /api/sightings/find-index/moose

Find the index of the moose sighting
GET /api/sightings/recent

Retrieve the three most recent sightings
